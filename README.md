# high_perf_feed
# Peak Feed - High Performance Social Feed

## Features
- Infinite scrolling (Supabase pagination)
- Pull-to-refresh
- Optimized UI using RepaintBoundary
- Memory optimization using cacheWidth
- Optimistic UI for like button
- Offline handling with UI revert
- Hero animation with tiered image loading

## Tech Stack
- Flutter
- Riverpod
- Supabase (PostgreSQL)

## Performance Optimizations
- RepaintBoundary prevents unnecessary GPU repaint
- cacheWidth reduces memory usage
- Batched pagination improves performance
- Thumbnail loading for faster rendering
